## :dart: About ##

Project developed with knowledge acquired in DevClub classes.
API of a hamburger shop, with order creation, list orders, update order and delete order.
It can also check its status by customer id.

## :rocket: Technologies ##

The following tools were used in this project:

- [Node.js](https://nodejs.org/en/)

## :white_check_mark: Requirements ##

Before starting :checkered_flag:, you need to have [Git](https://git-scm.com) and [Node](https://nodejs.org/en/) installed.
